# Vent MRP MVP

Streamlit tabanli bir is emri ve MRP (malzeme ihtiyac planlama) uygulamasi.

## Calistirma (Gelisim Ortami)
1) Python 3.11 kurulu olmali
2) Bagimliliklari kur:
   - pip install -r requirements.txt
3) Uygulamayi calistir:
   - streamlit run streamlit_app.py

## Tek tik calistirma (Paketli Surum)
CI uzerinden Windows icin .exe, macOS icin .app uretilir.
